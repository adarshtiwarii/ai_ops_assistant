## Key Design Principles

1. **Separation of Concerns**: Each agent has one job
2. **Modularity**: Easy to add new tools/agents
3. **Error Resilience**: Graceful failures, retries
4. **Type Safety**: Type hints throughout
5. **Observability**: Verbose logging for debugging
6. **Extensibility**: Plugin architecture for tools
